# Progetto di Tesi Triennale — Ingegneria Biomedica
Questo repository contiene i file sorgente del progetto sviluppato nell'ambito della tesi triennale in Ingegneria Biomedica. La tesi completa è disponibile nel file `tesi.pdf`.

## Requisiti
Per compilare ed eseguire il progetto è **fortemente consigliato** installare l'IDE **Qt**, che semplifica notevolmente l'intero processo di build. Per maggiori dettagli vedere la **sezione 2.5, Requisiti per la compilazione e sorgenti** della tesi.

## Compilazione ed esecuzione
1. Aprire l'IDE Qt
2. Caricare il file `.pro` presente nella cartella del progetto
3. Qt gestirà automaticamente l'intera toolchain di compilazione
4. Al termine della compilazione, l'applicazione si avvierà mostrando la schermata principale

## Demo senza hardware
È possibile testare parte del funzionamento dell'applicazione anche **senza la scheda di acquisizione dati**. Per simulare i dati in ingresso:
1. Avviare l'applicazione seguendo i passi descritti sopra
2. Premere il pulsante **"Play demo"** nell'interfaccia
Questo permetterà di visualizzare il funzionamento dell'applicazione con dati simulati.
